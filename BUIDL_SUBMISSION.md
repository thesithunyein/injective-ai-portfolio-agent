# BUIDL Submission Text

## Project Name
Casper AI Portfolio Agent

## One-Liner
An adorable, conversational AI portfolio manager for the Casper Network that pays for its own analysis via x402 micropayments.

## Problem
DeFi portfolio management is intimidating. Most tools are dark, complex, and require deep blockchain knowledge. There are no friendly, approachable AI agents for the Casper ecosystem that can analyze portfolios, answer questions in natural language, and operate autonomously.

## Solution
A cute, conversational AI agent (with a bear mascot!) that:
1. Fetches your Casper portfolio via CSPR.cloud API
2. Lets you chat with an AI agent about your holdings ("Should I buy more CSPR?")
3. Pays for AI analysis via Casper's x402 micropayment protocol
4. Stores results on-chain via an Odra smart contract
5. Gives you actionable rebalancing recommendations from Claude AI

## Key Features
- **Agent Chat** — Conversational AI that understands context and takes actions
- **x402 Integration** — HTTP-native micropayments (0.01 CSPR per analysis)
- **Smart Contract** — Odra-based upgradeable contract stores analysis on Testnet
- **Cute UI** — Pink accents, bear mascot, floating animations — stands out from generic DeFi apps
- **Real Data** — CSPR.cloud REST API for live Casper Testnet balances

## Tech Stack
- Next.js 14 + TypeScript + Tailwind CSS
- Claude 3.5 Sonnet API
- casper-js-sdk + CSPR.cloud API
- Odra Framework (Rust) smart contract
- x402 Micropayment Protocol
- Zustand state management

## What's New
- First cute-themed agentic AI on Casper
- x402 payment integration for autonomous agents
- Conversational portfolio management (not just dashboards)
- On-chain analysis storage via smart contract
